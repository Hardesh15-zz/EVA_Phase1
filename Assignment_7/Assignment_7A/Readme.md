## Calculation for RF # 224 * 224
N(output) = ( ( N(input) + 2 * (Padding) - K )/2 ) + 1 <br>
RF(output) = RF(input) + (K-1) * Jump(input) <br>
Jump(output) = Jump(input) * Stride
